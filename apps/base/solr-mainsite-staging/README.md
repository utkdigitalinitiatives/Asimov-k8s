# solr-mainsite-staging base — retired (July 6, 2026)

**This directory is intentionally empty**, kept as a placeholder so the
historical layout stays visible in the repo.

The base manifests for the staging Solr instance (namespace, SolrCloud with the
`/mainsite-staging` chroot) were removed when Solr moved to a production-only,
deploy-only model. The production base lives in
[`apps/base/solr-mainsite`](../solr-mainsite).

See the repo root `CLAUDE.md` for the current architecture.
